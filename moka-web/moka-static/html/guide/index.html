Introduction

Mobile first, responsive.

HTML5 doctype

모든 HTML 페이지에 시작 지점에 공백 없이 HTML5 문서 타입을 선언.

<!doctype html>
<html lang="ko">
  ...
</html>

html-charset

문서 루트인 <html> 요소에 lang 속성 추가.

<html lang="ko">
  ...
</html>

영어 : en 한국어 : ko 일본어 : ja

html-charset

문자열 인코딩을 명시적으로 선언.

<meta charset="utf-8">

html-ie-compatible

인터넷 익스플로러가 항상 최신 버전의 레이아웃 엔진을 사용하여 문서를 렌더링하도록 지정.

<meta http-equiv="X-UA-Compatible" content="IE=edge">

CSS

css는 <head></head> 내에 선언하는 것을 원칙.

<link href="common.css" rel="stylesheet">

CSS 내에 @import 는 성능을 저하시킵니다. 여러 개의 CSS 를 link.

JS

script는 <head></head> 내에 선언하는 것을 원칙으로하며 단, 성능상의 이슈가 있으면 개발 부서와 협의하여 선언 위치를 변경 가능.

<script src="common.js"></script>

Responsive meta tag

<!-- Adaptive meta tag -->
<meta name="viewport" content="width=device-width">

<!-- Responsive meta tag -->
<meta name="viewport" content="width=device-width, initial-scale=1">

<!-- Responsive meta tag / 브라우저 확대 및 축소를 비활성화 / 접근성 문제 -->
<meta name="viewport" content="width=device-width, initial-scale=1, minimum-scale=1.0, maximum-scale=1.0, user-scalable=no">

Meta tag

<!-- default meta tag -->
<meta name="title" content="title">
<meta name="description" content="description">
<meta name="keywords" content="keyword, keyword, keyword">

<!-- OpenGraph -->
<meta property="og:title" content="title">
<meta property="og:url" content="url">
<meta property="og:image" content="...png">
<meta property="og:description" content="description">
<meta name="twitter:card" content="summary">
<meta name="twitter:title" content="title">
<meta name="twitter:url" content="url">
<meta name="twitter:image" content="...png">
<meta name="twitter:description" content="description">
